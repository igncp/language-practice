## Meaning

### Git

- 位於分支：At branch （for git）
- 新檔案：New file
- 要提交的變更：Changes to be submitted
- 修改：Modified，revised

### Vim

- 頂端: Top (regarding the scroll, in bottom right)
- 全部：All (regarding the whole document is visible, in bottom right)
